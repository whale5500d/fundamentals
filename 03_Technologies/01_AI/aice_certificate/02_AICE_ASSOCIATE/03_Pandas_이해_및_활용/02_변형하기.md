# 2. Pandas 이해 및 활용 - 변형하기

## Groupby

- 범주형 컬럼을 기준으로 같은 값을 묶어서 통계 또는 집계 결과를 얻기 위해 사용하는 함수
- 예시
  - 성별 컬럼에서 F만 모으고, M만 모아서 F 그룹과 M 그룹 만들기
- 동작 원리: 데이터 분할(split), 적용(applying), 병합(combine) 순
- 연속형 컬럼의 평균, 힙계 등의 특징을 한 눈에 알아볼 수 있어 자주 사용하게 됨

## pivot_table

- 데이터프레임의 형태를 변경하는 함수
- 엑셀의 피봇 테이블과 동일
- 인덱스에 표시될 데이터와 컬럼에 표시될 데이터 모두 범주형 데이터여야 함.
  - 그래야 각 범주에 맞게 계산할 수 있기 때문
- `pandas.pivot_table(data, index, columns, aggfunc)
  - aggfunc: 집계 함수를 무엇으로 할 것인지 결정하는 인자
    - 예시: `pd.pivot_table(data=sample, index='고객ID', columns='상품코드', values='구매금액', aggfunc='mean')`

## stack, unstack

- stack(): 컬럼 레벨에서 보이는 컬럼명들을 인덱스 레벨로 가져와 데이터프레임을 변경
- unstack(): 인덱스 레벨에서 보이는 인덱스값을 컬럼 레벨로 가져와 데이터프레임을 변경
- 이렇게 데이터프레임을 변경해서 새로운 데이터를 뽑을 수 있는 퓨처 엔지니어링으로 사용할 수 있기에 알아두면 많은 도움

## Key Point

1. 그룹화(groupby())
   - 범주형 컬럼을 기준으로 같은 값을 하나로 묶어서 통계 또는 집계 결과를 얻기 위해 사용
2. 피봇테이블(pivot_table())
   - 데이터프레임 형태를 변경하는 것
   - 문법: pandas.pivot_table(data, index, columns, aggfunc)
3. 스택, 언스택(stack(), unstack())
   - Stack 함수는 컬럼 명들을 인덱스 레벨로 가져오는 것
   - Unstack 함수는 인덱스 값을 컬럼 레벨로 보내는 것
