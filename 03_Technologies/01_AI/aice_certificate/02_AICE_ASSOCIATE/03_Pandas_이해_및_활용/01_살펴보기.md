# 2. Pandas 이해 및 활용

## DataFrame 기본 함수

- head(): 앞에 5개 데이터를 보여줌
- tail(): 뒤에 5개 데이터를 보여줌
- shape(): row와 columns의 개수를 튜플로 반환함
- columns(): 컬럼명을 확인할 수 있음
- info(): 데이터 타입, 각 아이템의 개수 등을 출력함
- describe(): 데이터 요약 통계량을 나타냄
  - 기술적 통계 분석이 가능함.
  - count, mean, std, min, 25%, 50%, 75%, max

### 1. 원하는 column만 선택하기

- 리스트([])를 활용하여 column 추출
- 복수 개의 column을 가지고 오고자 한다면 리스트 활용

### 2. 슬라이싱(Slicing)

- 예시: `cust[7:10]`
  - `cust[7]` -> 에러 발생(column에서 찾기 때문, column은 숫자가 없는 상태)
  - `cust['base_yn'] -> 정상 실행(column은 'base_yn'과 같은 문제가 있으므로 정상 실행)
- row 레벨로 데이터를 가져올 수 있음.

### 3. Data 조회

- 데이터를 조회하는 두 가지 방법: loc, iloc
- loc
  - DataFrame의 **인덱스 번호, 컬럼 이름**을 기준으로 슬라이싱, 인덱싱 사용
  - DataFrame의 행이나 열에 label로 접근
  - location의 약자, **인간**이 읽을 수 있는 label 값으로 데이터 접근
- iloc
  - DataFrame의 **행이나 컬럼의 절대 번호**를 기준으로 슬라이싱, 인덱싱 사용
  - DataFrame의 행이나 열에 인덱스 값에 접근
  - integer location의 약자, **컴퓨터**가 읽을 수 있는 indexing 값으로 데이터 접근

### 4. Boolean Indexing

- DataFrame에서 원하는 행들만 추출 시 사용

### 5. Data 변경

- column 추가, 삭제하기
- 빈번하게 발생하는 작업
- 추가 예시
  - `cust['r3m_avg_bill_amt2'] = cust['r3m_avg_bill_amt'] * 2`
- 삭제 예시
  - `cust.drop('r3m_avg_bill_amt3', axis=1)`
  - axis가 0이면 세로 방향, axis가 1이면 가로 방향

  1. 판다스 데이터프레임
  - 판다스: 데이터 분서고과 데이터 전처리 시에 가장 많이 사용
  - 데이터프레임: 행과 열의 2차원 테이블 데이터 구조

  2. 데이터프레임 생성
  - 데이터프레임 생성: 딕셔너리, 리스트 활용

  3. 데이터프레임 조회
  - 데이터프레임 정보: head, tail, info, describe 함수
