# Pandas 이해 및 활용 - 병합하기

## concat/merge 함수

- 두 개 이상의 데이터프레임을 하나로 합치는 경우에 사용
- concat(): 동일한 컬럼명을 가지는 데이터프레임끼리 합치는 경우 주로 사용
- merge(): 두 데이터프레임을 공통된 컬럼을 기준으로 합치는 경우 사용
  - 4가지 방식: inner, left, right, outer
  - 예시: `pandas.merge(A, B, how='inner', on='item_id')`

## Key Point

1. 병합 함수
   - 두 개 이상의 데이터프레임을 하나로 합치는 경우 사용
   - concat(), merge() 함수가 대표적
2. concat()
   - 동일한 컬럼명을 가지는 데이터프레임끼리 합치는 경우 사용
   - axis 옵션에 따라 위아래 / 오른쪽왼쪽 방향으로 합침
3. merge()
   - 두 데이터프레임을 공통된 컬럼을 기준으로 합치는 경우 사용
   - How 옵션에 따라, inner, outer, left, right 방식이 존재
