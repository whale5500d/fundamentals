# 딥러닝 개념과 기술 원리

## 딥러닝 기본 개념

- 딥러닝 목표
  - 모델에 입력값을 넣었을 때의 출력값이 최대한 정답과 일치하게 하는 것
  - 머신러닝의 목표와 동일

- 딥러닝 학습 방법
  - 딥러닝 모델의 매개변수(weight, bias)를 무작위로 부여한 후, 반복학습을 통해 모델의 출력값을 정답과 일치하도록 매개변수를 조금씩 조정함.
  - Gradient Descent 최적화 알고리즘

## 딥러닝 기술 원리

1. Perceptron(퍼셉트론) (Node)
   - 사람 두뇌에 있는 뉴런을 모델링한 것
   - 간단한 함수를 학습할 수 있음.

2. 심층신경망(DNN, Deep Neural Network)
   - 입력층과 출력측 사이에 여러 개의 은닉층(hidden layer)으로 이루어진 인공신경망
   - 신경망 출력에 **비선형 활성화 함수**를 추가하여 복잡한 비선형 관계를 모델링 할 수 있음

3. Activation Function (활성화 함수)
   1. Binary Step
   2. Logistic, sigmoid, soft step
   3. Hyperbolic tangent(tanh)
   4. Rectified linear unit(ReLU)
      - ReLU, SELU, ELU, LReLU

4. 손실 함수 (Loss Function)
   - 신경망 학습의 목적함수로 모델의 예측갑소가 정답의 차이를 계산
   1. 회귀(Regression)
      - MSE, MAE
   2. 분류(Classification)
      - 이진분류(Binary cross-entropy)
      - 다중분류(Categorical cross-entropy)

5. Gradient Descent
   - DNN 모델의 가중치 파라미터들을 통해 경사 하강법으로 최적화 실행
   - 손실함수의 현 가중치에서 기울기(Gradient)를 구해서 Loss를 줄이는 방향으로 업데이트 실행

6. 역전파(Back Propagation)
   - 실제값과 모델 결과값에서 오차를 구해 오차를 output에서 input 방향으로 보냄
   - 가중치를 재업데이트 하면서 학습

7. 순전파(Forward Propagation)
   - 딥러닝 모델에 값을 입력해서 출력을 얻는 과정

8. 최적화 알고리즘 (Optimization Algorithm)
   - 딥러닝 모델의 매개변수(weight, bias)를 조절해서 손실함수의 값을 최저로 만드는 과정
   - 경사하강법(Gradient Descent)가 대표적이며, 대부분 Adam을 사용하면 됨.
   1. **GD** - 모든 자료를 다 검토해서 내 위치의 산 기울기를 계산해서 갈 방향을 찾겠다.
   2. **SGD** - 전부 다 봐야 한 걸음은 너무 오래 걸리니까 조금만 보고 빨리 판단한다. 같은 시간에 더 많이 간다.
   3. 스텝 방향
      1. Momentum - 스텝 계산해서 움직인 후, 아까 내려오던 관성 방향 또 가자.
      2. NAG(Nesterov Accelerated Gradient) - 일단 관성 방향 먼저 움직이고, 움직인 자리에 스텝을 계산하니 더 빠르더라.
      3. **Adam** - RMSProp + Momentum 방향도 스텝 사이즈도 적절하게
      4. Nadam - Adam에 Momentum 대신 NAG를 붙이자.
   4. 스텝 사이즈
      1. Adagrad - 안 가본 곳은 성큼 빠르게 걸어 훑고 많이 가본 곳은 잘 아니까 갈수록 보폭을 줄여 세밀히 탐색
      2. RMSProp - 보폭을 줄이는 건 좋은데 이전 맥락 상황을 봐가며 하자.
      3. AdaDelta - 종종걸음 너무 작아젹서 정지하는 걸 막아보자.
      4. Adam - 상기 동일

9. Data Set(데이터 셋)
   1. 1 Epoch: 모든 데이터 셋을 한 번 학습
   2. 1 Epoch = 10 \* batch_size
   3. SGD: batch size마다 순전파/역전파 학습 &rarr; 가중치 업데이트
