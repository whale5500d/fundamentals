# 딥러닝 주요 알고리즘

## 심층 신경망 (DNN, Deep Neural Network)

1. DNN 구현
   1. DNN 모델 Layout
      - Input Layer &rarr; 여러 개의 Hidden Layers &rarr; Ouput Layer
      - Output Layer는 한 개의 Unit으로 되어 있고, 한 개의 예측값을 출력
   2. Dropout
      - 과적합(Overfitting) 방지용
      - Train 학습 시에만 사용 (Test에는 반영되지 않음)

   3. 코드 예시

      ```
      // 라이브러리 임포트
      import tensorflow as tf
      from tensorflow.keras.models import Sequential
      from tensorflow.keras.layers import Dense, Dropout

      // 모델 생성 및 트레이닝
      model = Sequential()
      model.add(Dense(4, input_shape=(3,), activation='relu')) // 첫 번째 Hidden Layer
      model.add(Dropout(0.2))
      model.add(Dense(4, activation='relu')) // 두 번째 Hidden Layer
      model.add(Dense(1, activation='sigmoid')) // Output Layer

      model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])
      history = model.fit(X_train, y_train, validation_data=(X_test, y_test), epochs=40, batch_size=10)
      ```

## Key Point

1. 딥러닝 기본 개념
   - 머신러닝과 마찬가지로 모델에 입력값을 넣었을 때의 출력값이 최대한 정답과 일치하게 만드는 것
2. 딥러닝 용어
   - 퍼셉트론, 활성화 함수, 손실 함수, 순전파, 역전파
   - Epochs, batch_size, DNN 레이아웃
3. 딥러닝 코딩 템플릿
   - DNN 레이아웃에 맞게 딥러닝 코딩이 템플릿화 및 활용
   - Sequential > Dense > Compile > fit 순으로 진행
