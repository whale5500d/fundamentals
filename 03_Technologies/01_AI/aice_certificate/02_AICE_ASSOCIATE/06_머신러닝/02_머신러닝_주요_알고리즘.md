# 머신러닝 주요 알고리즘

## Logistic Regression

- 로지스틱 함수를 사용하여 결과값을 0~1 사이의 값으로 변환하여 이진 분류를 할 수 있는 알고리즘

```
from sklearn.linear_model import LogisticRegression
model = LogisticRegression()
model.fit(X_train, y_train)
pred = model.predict(X_test)
```

## K-Nearest Neighbor

- 새로운 데이터가 주어졌을 때 기존 데이터 가운데 가장 가까운 k개 이웃의 정보로 새로운 데이터를 예측하는 방법론
- 알고리즘이 간단하며 큰 데이터셋과 고차원 데이터에 적합하지 않음

```
from sklearn.neighbors import KNeighborsClassifier
model = KNeighborsClassifier(n_neighbors = 3)
model.fit(X_train, y_train)
pred = model.predict(X_test)
```

## Decision Tree

- 분류와 회귀 작업이 가능한 다재다능한 머신러닝 알고리즘
- 복잡한 데이터셋도 학습할 수 있으며,
- 강력한 머신러닝 알고리즘인 랜덤 포레스트의 기본 구성 요소

```
from sklearn.tree import DecisionTreeClassifier
model = DecisionTreeClassifier(max_depth = 2)
model.fit(X_train, y_train)
pred = model.predict(X_test)
```

## Random Forest

- 일련의(여러 개의) 예측기(분류, 회귀모델)로부터 예측을 수집하면 가장 좋은 모델 하나보다 더 좋은 예측을 얻을 수 있음
- 일련의 예측기는 앙상블, 결정 트리의 앙상블은 랜덤 포레스트
- 훈련 세트로부터 무작위로 각기 다른 서브셋을 만들어 일련의 결정 트리 분류기를 훈련시킬 수 있음

```
from sklearn.ensemble import RandomRorestClassifier
model = RandomRorestClassifier(n_estimators = 50) // 랜덤 포레스트에서 사용할 내부 Decision Tree 개수
model.fit(X_train, y_train)
pred = model.predict(X_test)
```

- Random Forest의 주요 하이퍼 파라미터
  - n_estimaters: 생성할 트리의 개수 (int, default = 100)
  - max_depth: 트리의 최대 깊이 (int, default = None)
  - min_samples_split: 노드를 분할하기 위해 필요한 최소한의 샘플 개수 (int or float, default = 2)

- Random Forest의 변수중요도 속성
  - feature_importances: 변수에 대한 중요도 값을 제공

# 하이퍼파라미터 튜닝 도구

## GridSearchCV

- 하이퍼파라미터를 순차적으로 입력해 학습을 하고 측정하면서 가장 좋은 최적의 파라미터를 알려준다.

```
from sklearn.model_selection import GridSearchCV
model_1 = RandomForestClassifier()
params = { 'n_estimators': [100, 150], 'max_depth': [2, 5] }
model_2 = GridSearchCV(model_1, param_grid=params)
model.fit(X_train, y_train)
pred = model.predict(X_test)
```

- GridSearchCV의 중요 속성
  - grid_rfc.best_params\_: 최적의 파라미터 리스트
  - grid_rfc.best_score\_: 최적의 파라미터일 때의 Score

## Key Point

1. 머신러닝 템플릿
   - 머신러닝 템플릿: 모델 import, 모델 정의, 모델 학습, 모델 예측
   - 머신러닝 모델 만들고 학습하는데 4~5줄이면 됨
2. 지도학습 모델
   - 회귀 모델: LinearRegression, Decision Tree, RandomForest
   - 분류 모델: LogisticRegression, Decision Tree, RandomForest
3. 성능 최적화
   - GridSearchCV 활용하여 모델의 최적의 파라미터 선별
   - best_params\_, best_score\_ 속성 정보로 최적의 파라미터와 점수 확인
