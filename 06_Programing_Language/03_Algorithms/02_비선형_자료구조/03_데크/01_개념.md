# 데크(Deque)

- Double-Ended Queue의 약자
- 삽입과 삭제가 양쪽 끝에서 모두 발생할 수 있는 선형 자료 구조
- 구현 메서드(Method)
  - 데이터 전체 획득/비어 있는지 확인: Deque.getBuffer(), Deque.isEmpty()
  - 추가/삭제
    - 추가: Deque.pushFront(), Deque.pushBack()
    - 삭제: Deque.popFront(), Deque.popBack()
  - 첫번째 & 끝 데이터 반환: Deque.front(), Deque.back()
  - 사이즈: Deque.size()
  - 전체 삭제: Deque.clear()
