# 큐(Queue)

- 먼저 넣은 데이터가 먼저 나오는 FIFO 기반의 선형 자료 구조
- 구현 메서드(Method)
  - 데이터 전체 획득: Queue.getBuffer()
  - 비어있는지 확인: Queue.isEmpty()
  - 추가/삭제: Queue.enqueue(), Queue.dequeue()
    - 추가: push와 동일, 뒤에서 추가
    - 삭제: pop과 다름, 앞에서 삭제
  - 첫번째 데이터 조회: Queue.front()
  - 크기 확인: Queue.size()
  - 전체 삭제: Queue.clear()
