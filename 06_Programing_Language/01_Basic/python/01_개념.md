# 1. Python이란 무엇인가

## 1.1. Python이란?

- 파이썬은 컴퓨터 프로그래밍 교육을 위해 많이 사용되고, 기업의 실무에서도 많이 사용되는 언어
- 구글, 인스타그램, 드롭박스 등

## 1.2. Python의 특징

- 공동 작업과 유지 보수가 매우 쉽고 편하다. 그 때문에 이미 다른 언어로 작성된 프로그램과 모듈이 파이썬으로 재구성되고 있다.

### 1. 파이썬은 무료이지만 강력하다.

- 프로그래머는 만들고자는 프로그램의 대부분을 파이썬으로 만들 수 있다.
- 물론 시스템 프로그래밍이나 하드웨어 제어와 같은 복잡하고 반복 연산이 많은 프로그램은 파이썬과 어울리지 않는다.
- 하지만 이러한 약점을 극복할 수 있게끔 다른 언어로 만든 프로그램을 파이썬 프로그램에 포함시킬 수 있다.
- 파이썬과 C는 찰떡궁합. 즉 프로그램의 전반적인 뼈대는 파이썬으로 만들고, 빠른 실행 속도가 필요한 부분은 C로 만들어서 포함시킨다.

### 2. 파이썬은 간결하다.

- 펄(Perl)과 같은 프로그래밍 언어는 100가지 방법으로 하나의 일을 처리할 수 있다면, 파이썬은 가장 좋은 방법 1가지만 사용하는 것을 선호한다.
- 이 간결함의 철학은 파이썬 프로그래밍을 하는 사람들은 잘 정리되어 있는 소스 코드를 볼 수 있다. 다른 사람이 작업한 소스코드도 한 눈에 들어와 이해하기 쉽기 떄문에 공동 작업과 유지 보수가 아주 쉽고 편하다.

## 1.3. 파이썬으로 할 수 있는 작업

1. 시스템 유틸리티 제작
2. GUI 프로그래밍
3. C/C++와의 결합
4. 웹 프로그래밍
5. 수치 연산 프로그래밍
6. 데이터베이스 프로그래밍
7. 데이터 분석, 사물 인터넷

## 1.4. 파이썬으로 할 수 없는 작업

1. 시스템과 밀접한 프로그래밍 영역
2. 모바일 프로그래밍
