# 2. 자료형

어떤 프로그래밍 언어든 '그 언어의 자료형을 알고 이해할 수 있다면 이미 그 언어의 절반을 터득한 것이나 다름없다.'는 말이 있다.
자료형이란 프로그래밍을 할 때 쓰이는 숫자, 문자열 등 자료 형태로 사용하는 모든 것을 뜻한다. 프로그램의 기본이자 핵심 단위가 자료형이다.
계산 프로그램을 만들려면 어떤 것을 계산할지 알아야 하고, 데이터베이스 프로그램을 만들려면 어떤 자료를 저장할지부터 알아야 하는 것과 같다.

## 1. 숫자형

- 숫자형은 정수, 실수, 8진수, 16진수로 구성된다.
- 연산자는 +, -, \*, /, \*\*, %(나머지), //(몫)

## 2. 문자열 자료형

- 따옴표 사용
  ```
      // 단줄 입력: "___", '___'
      // 여러 줄 입력: """___""", '''___'''
  ```
- 문자열 연산
  1. 문자열 더하기
     ```
         head = "Python"
         tail = " is fun!"
         head + tail
         // return: 'Python is fun!'
     ```
  2. 문자열 곱하기
     ```
         a = "Python"
         a * 2
         // return: 'PythonPython'
     ```
  3. 문자열 길이 구하기
     ```
         a = "Life is too short"
         len(a)
         // return: 17
     ```
- 문자열 인덱싱과 슬라이싱
  - 인덱싱
    - 0부터 앞에서 인덱스 사용
    - -1부터 뒤에서 인덱스 사용
    - -0은 0과 동일한 인덱스 사용
  - 슬라이싱
    - `a[0:4]` (0 <= a < 4)
    - 시작번호, 끝번호 중 하나는 생략할 수 있다. 생략 시 처음, 마지막 인덱스부터 슬라이싱한다.
    - 예시
      ```
          a = "Life is too short, You need Python"
          a[19:-7]
          // return: "You need"
          // a[19]부터 a[-8]까지 의미한다. a[-7]은 포함하지 않는다.
      ```

- 문자열 포멧팅
  |코드|설명|
  |--|---|
  |%s|문자열(String)|
  |%c|문자 1개(Character)|
  |%d|정수(Integer)|
  |%f|부동 소수(Floating-point)|
  |%o|8진수|
  |%x|16진수|
  |%%|Literal % (문자 '%' 자체)|
