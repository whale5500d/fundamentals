# Summary - Multivariable Calculus

1. **편미분과 gradient**: 다변수 함수(변수가 여러 개인 함수)에서 하나의 변수만 변화시켰을 때 함수값이 얼마나 변하는지 측정하는 것이 편미분임. gradient는 모든 변수에 대한 편미분을 모은 벡터로, 함수값이 가장 가파르게 증가하는 방향을 나타냄.
2. **chain rule**: 함수가 여러 단계로 합성되어 있을 때 전체 미분을 각 단계 미분의 곱으로 분해하는 규칙임. 신경망의 backpropagation은 이 규칙을 다층 구조에 반복 적용해 각 가중치에 대한 gradient를 계산하는 것과 동일한 구조임.
3. **Jacobian, Hessian**: Jacobian은 다변수 함수의 출력이 여러 개일 때 편미분을 행렬로 표현한 것임. Hessian은 2차 편미분을 행렬로 표현한 것으로, 함수의 곡률(curvature) 정보를 담고 있어 최적화 알고리즘의 수렴 방향 분석에 사용됨.
4. **Gradient Descent**: gradient가 가장 가파르게 증가하는 방향의 반대 방향으로 조금씩 이동하며 함수의 최솟값을 찾아가는 반복 알고리즘임. 신경망 학습의 핵심 메커니즘이며, 갱신 규칙 θ ← θ − η∇L(θ)가 이를 표현함.
5. **Lagrange multiplier(라그랑주 승수법)**: 등호 제약 조건(예: 특정 조건을 만족해야 함)이 있는 최적화 문제를 푸는 수학적 기법임. AI에서는 SVM(서포트 벡터 머신)의 마진 최대화, 정보 이론의 최대 엔트로피 문제 등에 등장함.
