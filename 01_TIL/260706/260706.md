### 4교시

- Affine
  - 모델 ID에 AWQ, GPTQ와 같은 접미사가 붙으면 양자화가 되어있는 파일을 받아서 사용
  - 모델의 Loading이란: 허깅 페이스로부터 모델 파일을 내려받는다. (FP16)
  - 모델의 가중치를 파일에서 비디오 메모리로 올린다.(다운 받는다고 해서 가중치가 그대로 사용할 수 있는 게 아님) (INT4, 양자화 실행)
    - HBM에 있는 정보를 DDR로 옮긴다. (느린 메모리로 옮김) -> offloading
    - 반대는 onloading

- Weight Quantization
  - Per-tensor, Per-channel, Per-group 가중치 양자화

- Activation Quantization
  - 하지만 가중치만 양자화하면, 추론 중 생성되는 활성화 값은 여전히 FP16이나 BF16 같은 부동소수점 형식으로 남는 경우가 많다.
  - 대표적인 방식
    - 대표 데이터를 미리 넣어 활성화 분포를 측정하고, 그 결과 Scale, Zero-point를 추론 시 계속 사용하는 방식
    - 추론 시점에 입력별로 활성화 범위를 측정하고, 그때마다 Scale을 계산하는 방식
    - DRAM, VRAM으로 떄우고 있음

- KV Cache Quantization
  - 그냥 저장하지 않고, 이전 토큰의 Key, Value 벡터를 저비트로 저장

- idea
  - 자기 모델, 작은 모델 -> 금칙어 판정(가드레일) -> 생성이 아니라 판정
